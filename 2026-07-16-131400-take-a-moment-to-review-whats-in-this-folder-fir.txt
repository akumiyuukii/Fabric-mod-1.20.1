  Project overview

  A Fabric Minecraft mod by "AkumiYuukii" (mod id akumiyuukiimods). Despite the
  folder name fabric-1.21.11-mod, it actually targets Minecraft 1.20.1
  (gradle.properties, fabric.mod.json), Java 21, Fabric Loader 0.19.3 + Fabric
  API 0.92.9.

  Two things the mod does:
  1. A stat / combat system — HP, Attack, Defense, Speed, Jump stats, a custom
  damage_boost effect and bonus_damage enchantment, damage tracking, and a
  damage HUD overlay. Registered in AkumiYuukiiMods.java.
  2. An in-game AI chat client — a settings screen + chat screen that POST to a
  local AI server (http://localhost:3000/api/components/chat/v1), with
  connect-test, model-scan, and log features (FabricAIBridge.java,
  AISettingsScreen, AIChatScreen). UI text is in Vietnamese.

  Layout

  build.gradle / gradle.properties / settings.gradle   ← Fabric Loom build (MC
  1.20.1)
  fix2.py, fix3.py                                      ← Python scripts that
  rewrite Java source
  src/main/java/akumiyuukii/mods/                       ← common: stats,
  enchant, effect, AI bridge, mixins
  src/client/java/akumiyuukii/mods/client/              ← client:
  chat/settings/stat screens, HUD, mixin
  src/.../com/example/...                               ← leftover Fabric
  example-mod template files

  - Leftover template cruft — com/example/... classes and modid.mixins.json /
